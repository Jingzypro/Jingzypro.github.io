Standard Search Algorithm Vs CPS Algorithm

The standard search algorithms treat the problems as a blind search over all possible assignments, while most of these assignments violate constraints. But they explore them anyway, wasting time and space.

CSP-specific algorithms are much faster because they use constraints to prune the search space early.

Using Backtracking Search for CSPs

Using DFS with single-variable assignments at each level is called backtracking search

Variable assignment is commutative

Improving Search Efficiency

General-purpose heuristics and Inference can dramatically improve search efficiency

  • Which variable should be assigned next?

    • Minimum Remaining Values(MRV) heuristic
    • Degree heuristic

  • In what order should the chosen variable’s values be tried?

    • Least Constraining Value heuristic

  • Using inference during search

    • Forward Checking
    • AC-3 Algorithms
Minimum Remaining Values(MRV) heuristic

Choose the variable with the fewest legal values left

Degree heuristic

Choose the variable with the most constraints on remaining (unassigned) variable, or that has the largest number of unassigned neighbours.

In the Backtracking Algorithm, first apply the MRV heuristic. If there are 2 or more variables left after applying MRV, apply the Degree heuristic.

Least Constraining Value heuristic

Order its domain values according to the number of values they rule out in the domain of the unassigned neighbour.

Forward Checking

After a variable is assigned a value, update the domain values of its neighbors (a neighbor is another variable that shares one or more constraints with the current variable.

Forward checking does not provide early detection for all failures. ->

AC-3 Algorithm

One of the Constraint Propagation is Arc Consistency

AC-3 is the most popular Arc Consistency algorithm

If X loses a value, the neighbors of X need to be rechecked

Overfitting

Overfitting happens when a model learns the training data too well, including noise and random patterns that are not truly general. It performs great on the training set but poorly on new unseen data.

Underfitting

The chosen model is too simple to find a pattern in the training data.

Generalization

Generalization is the ability of a model to perform well not only on the training set but also on new unseen data.

Types of Learning Methods

Supervised Learning

Learns from labeled data (input + correct output).

Unsupervised Learning

Learns from unlabeled data.

Learn from structures like clustering

Semi-supervised Learning

Uses a small set of labeled data together with a large set of unlabeled data.

Self-supervised Learning

Uses no human labels; the model generates its own training targets.

Reinforcement Learning

Learns by interacting with an environment and receiving rewards or penalties.

Type Label Example
Supervised Learning Yes Image classification, spam detection, price prediction

Information gain

Entropy H(V): a measure of the uncertainty of a random variable V
More information -> reduced randomness or entropy

For a random variable (V) with values (v_k), each with probability (P(v_k)),

(H(V) = \sum_{k} P(v_k) \log_2 \frac{1}{P(v_k)} = -\sum_{k} P(v_k) \log_2 P(v_k))
